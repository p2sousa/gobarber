## :information_source: Como Usar

Clone o repositório [GoBarber](https://github.com/p2sousa/gobarber):

```bash
# Clone this repository
$ git clone https://github.com/p2sousa/gobarber

# Go into the repository
$ cd backend

# Run Docker
$ docker-compose up -d

# Install dependencies
$ yarn install

# Run the app
$ yarn start
```
